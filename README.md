# Wanderly

Travel diary — trips, day-by-day photo memories, timeline and stats.
Single-file mobile PWA.

**Live:** https://philduggan88.github.io/wanderly/

Ships with seeded demo data; your own trips and photos live only in the
phone's browser storage (with JSON backup export/import) — nothing personal
is in this repo. On iPhone open the URL in Safari → **Share → Add to Home
Screen** for the full-screen offline app.

This repo is the built output only. The source app runs locally on my Mac;
this page is regenerated with its `mobile/build.py` and pushed here.
